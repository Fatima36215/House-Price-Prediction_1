# 🏠 House Price Prediction using Linear Regression

## 📌 Project Overview
This project predicts house prices using the California Housing Dataset and the Linear Regression algorithm.

## 📂 Dataset
- California Housing Dataset
- Source: Scikit-learn

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 🤖 Machine Learning Algorithm
- Linear Regression

## 📊 Model Performance
- MAE: 0.5332
- RMSE: 0.7456
- R² Score: 0.5758

## 📈 Project Workflow
1. Data Loading
2. Data Exploration
3. Missing Value Check
4. Correlation Analysis
5. Train-Test Split
6. Linear Regression Model
7. Model Evaluation
8. House Price Prediction

## 📌 Conclusion
The Linear Regression model successfully predicted house prices and achieved an R² score of approximately 57.6%. This project demonstrates the complete workflow of a basic machine learning regression model.
